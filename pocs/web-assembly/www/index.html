Aloca bytes no heap Rust via alloc(), escreve diretamente com fill_buffer(), lê de volta via wasm.memory.buffer. Zero cópias JS↔WASM.
Soma do buffer — leitura direta sem passar pelo JS

Rust chamando closures JavaScript

for_each_prime(limit, callback) — Rust itera os primos e chama uma função JS para cada um. Observe o overhead de cross-boundary calls.
reduce_primes(limit, reducer, initial) — aplica um reducer JS em todos os primos. Exemplo: somar, encontrar o maior, calcular produto.

Custo de strings na boundary WASM

Strings cruzam a boundary com alocação UTF-8 — muito mais caro que buffers numéricos. Compare count_occurrences WASM vs indexOf loop em JS.
reverse_string(s) — reversão correta de Unicode (por char, não byte)
