# What I learned?

- Document versioning
- How to document a component
- Docusaurus navbar configuration
- Docusaurus theme configuration

## Tech Details

- Docusaurus as site generator
- MarkdownX as docs format

## Key Modules:

<b>Button</b>: Component documentation with props, examples, and usage.

<b>ButtonNew</b>: Simulates a new version of Button component with new props.

## Execution

```bash
pnpm start
```

## Final Thoughts

Docusaurus is a nice way to generate documentation sites with versioning and component examples, allowing you to document your components with their changes and also compare then in the same page, I will use it in my next project, for sure.
